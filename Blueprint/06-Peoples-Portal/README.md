## Vision
The single interface for 30 Million Nepalese to interact with their government.

## Features
- **My Taxes:** See exactly where your tax money went (e.g., "Rs 500 to Melamchi Project").
- **Vote:** Secure, digital voting on local issues (Participatory Budgeting).
- **Report:** "Fix This Pothole" - geo-tagged reporting with status tracking.
- **Identity:** Self-Sovereign Identity (SSI) integration.

## UX Philosophy
- **Mobile First:** 90% of access will be via smartphones.
- **Offline Capable:** Must work in remote areas with spotty internet.
- **Multilingual:** Nepali, Maithili, Bhojpuri, English, etc.
