**"Code as Law"** - Where legislation becomes machine-readable and transparent.

## Scope

Covers all legislative functions across Nepal's federal structure:

### Federal Parliament
- **House of Representatives** (Pratinidhi Sabha) - 275 members
- **National Assembly** (Rastriya Sabha) - 59 members

### Provincial Assemblies
- 7 Provincial assemblies (total ~550 members)

## Features

- **Bill Version Control**: Git-like diffing of legislation amendments
- **Impact Analysis**: Automated analysis of how new laws affect existing systems
- **Voting Records**: Public record of all votes by elected representatives
- **Committee Proceedings**: Tracked discussions and decisions
- **Public Consultation**: Citizen input on proposed legislation

## Key Entities

| Entity | Description |
|--------|-------------|
| `Bill` | Proposed legislation with version history |
| `Act` | Passed legislation in force |
| `Amendment` | Changes to existing legislation |
| `Representative` | Elected member with voting record |
| `Committee` | Parliamentary/Provincial committee |
| `Session` | Legislative session records |

## Transparency Goals

- Every bill change is tracked with git-style diffs
- Citizens can comment on bills during public consultation
- AI-assisted impact summaries for complex legislation
- Representatives' voting patterns publicly accessible
