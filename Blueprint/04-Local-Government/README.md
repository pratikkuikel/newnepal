# üè† Local Government (Ward Level)

The grassroots tier. This is where government meets the citizen directly.

## Scope

Each municipality is divided into **Wards** (9-35 per municipality). Ward offices are the frontline of service delivery.

## Responsibilities

- **Citizen Documentation**: Birth/death certificates, ID verification
- **Community Programs**: Local development initiatives
- **Neighborhood Auditing**: Verification of project completion at ground level
- **Grievance Collection**: First point of contact for citizen complaints
- **Social Security Distribution**: Allowance distribution to elderly, disabled

## Key Entities

| Entity | Description |
|--------|-------------|
| `Ward` | Administrative subdivision of municipality |
| `WardOffice` | Physical ward administration |
| `CommunityProject` | Ward-level micro-projects |
| `CitizenRecord` | Resident registration and documentation |
| `GrievanceTicket` | Citizen complaint/request record |

## Citizen Touchpoints

- Birth/Death/Marriage registration
- Property ownership verification
- Voter registration assistance
- Social security card issuance
- Local dispute mediation
